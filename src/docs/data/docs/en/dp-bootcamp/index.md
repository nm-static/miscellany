# Dynamic Programming Bootcamp

**Year:** 2022
**Type:** Certification
**Instructor:** Priyansh Agarwal

## Overview

An intensive bootcamp focused on mastering dynamic programming techniques for competitive programming and interviews.

## Topics Covered

1. Introduction to DP
2. 1D DP Problems
3. 2D DP Problems
4. DP on Trees
5. Bitmask DP
6. DP Optimization Techniques

## Format

- Daily problem-solving sessions
- Live coding demonstrations
- Practice contests

## Prerequisites

- Basic programming skills
- Arrays and recursion fundamentals
