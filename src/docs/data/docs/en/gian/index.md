# GIAN Course: Randomized Methods for Parameterized Algorithms

**Year:** 2022
**Code:** 191014K02
**Instructor:** Prof. Daniel Lokshtanov

## Overview

This GIAN (Global Initiative of Academic Networks) course covers randomized techniques in the context of parameterized algorithms.

## Topics Covered

1. Introduction to Parameterized Complexity
2. Randomized Algorithms Basics
3. Color Coding
4. Random Separation
5. Algebraic Techniques

## Prerequisites

- Algorithms and Data Structures
- Basic Probability Theory
- Computational Complexity
